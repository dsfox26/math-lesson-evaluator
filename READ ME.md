# Math Lesson Plan Evaluator

AI-powered evaluation of K-12 mathematics lesson plans using research-based frameworks.

## Frameworks

1. **Task Analysis Guide** (Institute for Learning) - Cognitive demand of individual tasks
2. **Mathematical Quality of Instruction (MQI)** - Overall lesson quality
3. **Problem-Based Learning Principles** - Student-centered mathematics instruction

## Setup

1. Install dependencies: `npm install`
2. Add your Claude API key to `.env.local`
3. Run locally: `npm run dev`
